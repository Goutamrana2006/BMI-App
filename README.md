# ⚖️ BMI Calculator Web Application:

# 📌 Project Overview
The **BMI Calculator Web Application** is a responsive and interactive web-based tool that helps users calculate their **Body Mass Index (BMI)** using height and weight inputs. The application provides instant BMI results, health status classification, personalized health tips, BMI visualization, and history tracking for a better user experience.
This project was developed to strengthen frontend development skills and demonstrate practical implementation of JavaScript, DOM manipulation, Local Storage, and responsive UI design.

---
## 🚀 Features

✅ Calculate BMI instantly using height and weight
✅ Automatic BMI category detection:
* Underweight
* Normal Weight
* Overweight
* Obese
✅ Dynamic BMI meter with color indicators
✅ Personalized health recommendations
✅ Local Storage integration for BMI history
✅ Clear History functionality
✅ Modern Glassmorphism User Interface
✅ Animated and responsive design
✅ Interactive BMI status visualization

## 📊 BMI Categorie
| BMI Range    | Status        |
| ------------ | ------------- |
| Below 18.5   | Underweight   |
| 18.5 - 24.9  | Normal Weight |
| 25 - 29.9    | Overweight    |
| 30 and Above | Obese         |
